# VoiceScanAI-and-ScamVision

VoiceScanAI-and-ScamVision is an AI-powered mobile application designed to detect potential scam calls by analyzing both voice characteristics and conversation content in real time. The system combines speech processing, deep learning, and natural language processing techniques to identify suspicious calls and provide risk alerts to users during phone conversations.

## Features

* Real-time call monitoring and analysis
* Speech-to-Text transcription using Sherpa-ONNX Zipformer
* Scam content detection using TextCNN
* AI-generated voice detection using Conformer
* Audio spoofing detection using AASIST
* Multi-model score fusion for risk assessment
* Floating overlay displaying live detection results
* Continuous risk scoring throughout the call

## System Architecture

The application consists of three main detection modules:

### 1. Speech Recognition

Incoming call audio is processed using a Zipformer ASR model to generate real-time transcripts.

```text
Call Audio
    ↓
Zipformer ASR
    ↓
Transcript
```

### 2. Text Scam Detection

The generated transcript is analyzed using a TextCNN classifier trained to distinguish between normal conversations and scam-related content.

```text
Transcript
    ↓
TextCNN
    ↓
Scam Probability
```

### 3. Voice Analysis

Audio segments are simultaneously analyzed by two voice detection models:

* Conformer-based AI voice detector
* AASIST spoofing detector

```text
Audio Window
    ↓
 ┌─────────────┐
 │ Conformer   │
 └─────────────┘
         ↓
 ┌─────────────┐
 │  AASIST     │
 └─────────────┘
         ↓
 Voice Risk Score
```

## Risk Fusion

The final risk score is computed by combining text and voice analysis results.

```text
Text Detection
       ↓
Voice Detection
       ↓
 Detection Aggregator
       ↓
 Final Scam Risk
```

The fusion mechanism continuously updates the overall risk score and generates warnings when suspicious activity is detected.

## Research Objectives

This project aims to:

* Detect scam-related conversations through NLP techniques.
* Identify AI-generated or spoofed voices during calls.
* Provide real-time risk assessment for mobile users.
* Improve awareness and protection against modern voice-based fraud.

## Disclaimer

This application is designed as a research and educational project. Detection results are probabilistic predictions generated by machine learning models and should be considered as decision-support information rather than definitive proof of fraudulent activity.
